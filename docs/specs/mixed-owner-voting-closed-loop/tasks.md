## 1. 规则和实际方式

- [x] 仅允许有效规则明确配置 `ONLINE_AND_OFFLINE + PAPER_AND_ONLINE`；

- [x] 混合方式直接开放纸票和线上入口，不要求纸质协助申请；

- [ ] 被替代票、后到材料和线上冲突均保留完整审计；
- [ ] 验证并发下数据库唯一约束与冻结策略一致。

## 3. 投影和接端

- [ ] 盘古/瑶池仅在有效规则允许范围内选择本次方式，并提供纸票办理台。
