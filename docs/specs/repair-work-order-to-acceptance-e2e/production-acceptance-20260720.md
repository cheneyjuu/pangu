## 4. 前置条件复核

第一次生产演练在合同签署前识别出隔离租户的物业经理尚未挂接已启用物业项目部。
这不是工程状态问题，而是合同签约主体前置资料不完整。完成物业服务组织、项目部和物业身份挂接后，
重新从业主报修开始执行全流程并通过。

该问题泛化为发布验收规则：纵向 BDD 开始前必须一次性复核物业服务组织、项目部、物业工作身份、
楼栋专项维修资金账簿和业主实名等级；不得等到合同或表决环节再逐项暴露基础资料缺口。
