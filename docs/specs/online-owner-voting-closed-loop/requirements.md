# 线上实名投票闭环需求

> 业务关联：业主大会及维修相关业主共同决定中的互联网表决、实名资格校验、逐户确认、纸质协助和跨渠道唯一。
>
> 状态：实施中（2026-07-19）
>
> 上位业务规范：[维修与业主大会表决业务规范](../../business/维修与业主大会表决业务规范.md)
>
> 对应改造切片：[维修与表决能力改造计划 - 切片五](../../plans/维修与表决能力改造计划.md#切片五线上实名投票)

## 1. 范围

本切片在统一表决执行内核之上实现互联网表决渠道。线上提交必须绑定当前自然人、当前房屋表决权代表、本次冻结名册和已锁定表决包；一个专有部分的一次确认覆盖本次全部表决事项。平台不把登录、微信昵称、手机号或客户端传入的摘要当作表决人签名。

本切片同时提供上海示范文本要求的纸质协助申请。该申请不是第二场投票，也不改变本次规则；申请后，该专有部分通过已锁定的正式纸质表决票办理，并与线上票共用同一唯一约束。

## 2. 功能需求

### R1 单次实际方式来自有效规则

1. 本小区有效规则允许互联网会议且要求线上实名核验时，单次会议才可选择“互联网表决并提供纸质协助”；
2. 该方式仍须锁定正式纸质表决票模板，用于确有困难并提出要求的业主；
3. 规则只允许纸质时不得开启线上入口；规则未明确允许纸质与线上并行时，不得把纸质协助解释为向所有业主开放的并行渠道；
4. 本次实际方式、规则版本、表决时间和材料进入同一表决包摘要，发布后不可修改。

### R2 实名资格和本人房屋

1. 只有当前 C 端自然人账号通过活体实名核验后才能在线确认和提交；
2. 后端校验当前账号、自然人、租户、有效房屋关系和冻结表决代表，不接受前端传入他人账号、自然人或表决人标识；
3. 当前自然人有多个符合资格的专有部分时逐一办理，每个专有部分独立形成参与状态和回执；
4. 资格已失效、不在冻结名册、会次不属于当前小区或不在投票期时拒绝提交。

### R3 阅读确认和有效送达

1. 业主进入正式表决前必须看到本次公告、方案附件、表决事项、适用规则、截止时间和本人对应专有部分；
2. 业主明确确认已阅读本次锁定材料后，后端记录表决包摘要、专有部分、当前自然人、确认时间和确认记录摘要；
3. 该确认同时形成当前线上渠道的有效送达记录；未确认阅读不能在线提交；
4. 表决包摘要变化、会次被作废或确认人不再是冻结表决代表时，旧确认不能继续使用。

### R4 整包确认提交

1. 一次线上提交必须包含本次表决包全部表决事项，不能逐事项零散写入正式票台账；
2. 页面提交前须再次展示全部选择，并要求业主明确确认；
3. 后端校验事项集合与锁定表决包完全一致、选项有效、表决包摘要一致、阅读确认有效；
4. 全部事项在一个事务中写入统一有效票台账，任一事项失败则整次提交不产生部分有效票；
5. 成功后返回不可变的在线确认记录编号、提交时间和服务端摘要；摘要是审计校验值，不宣传为电子签名或数字证书。

### R5 幂等、重复和隐私

1. 同一专有部分只能完成一次有效提交；网络重试使用同一幂等键和相同内容时返回原回执，不重复计票；
2. 同一幂等键内容不同时拒绝；已有纸质有效票、另一笔线上有效票或并发冲突时拒绝并保留审计；
3. 提交成功后业主只能看到“已完成”和本人回执，不回显具体选择；
4. 管理端只显示本人办理所需的汇总进度和纸质协助请求，不向物业展示线上票面选择。

### R6 纸质协助申请

1. 互联网表决期间，冻结名册中的表决代表可为本人某一专有部分申请纸质协助，无需说明不会使用手机的原因；
2. 已完成有效线上投票后不能再申请；申请后线上提交入口关闭；
3. 经办人员只能为已申请的专有部分登记纸质送达和回收，使用现有纸质双人复核闭环；
4. 尚未登记纸质送达或回收前，业主可撤回申请并恢复线上入口；已有纸质办理事实后不得撤回；
5. 管理端展示待提供纸票、办理中和已完成，不使用“异常”“受阻”等技术状态。

## 3. 验收场景

1. 活体实名业主阅读并确认锁定材料后，系统形成线上送达；未确认阅读不能提交；
2. 一次提交覆盖全部表决事项并返回回执，数据库不存在只写入部分事项的状态；
3. 客户端伪造账号、自然人、专有部分、事项、表决包摘要或签名摘要均不能形成有效票；
4. 相同幂等键和相同内容重试返回同一回执；更换内容重试被拒绝；
5. 一个专有部分已经形成纸质有效票后，线上提交被拒绝；并发提交最多产生一组有效票；
6. 业主申请纸质协助后线上入口关闭，物业可按已锁定模板办理纸票；未申请的专有部分不能走纸质协助；
7. 业主端和物业端均看不到线上具体选择，只有有权计票和审计能力读取受控原始记录；
8. 旧通用单事项投票接口不能绕过整包确认流程写入正式业主大会票。

## 4. 非本切片范围

- 本小区有效规则明确允许向全体业主同时开放纸质与线上时的并行渠道工作台；
- 电子签章、CA 数字证书或第三方存证平台；
- 未反馈按多数或按弃权等推导规则；
- 历史线上汇总数据反推为逐户选票。
